# inton

웹페이지의 뉴스 링크 위에 마우스를 800ms 이상 올려두면 Groq 또는 Cerebras API로 기사 여부, 신뢰도, 어그로도를 분석해 커서 옆 팝업에 보여주는 Chrome Extension Manifest V3 프로젝트입니다.

## 1. 프로젝트 소개

이 확장 프로그램은 페이지 안의 `<a>` 링크를 감지하고, 사용자가 링크에 잠시 머무르면 링크 텍스트와 URL을 수집합니다. background service worker는 가능하면 해당 URL의 HTML을 가져와 `title`, `meta description`, `og:title`, `article`, `p` 태그 일부를 추출한 뒤 AI 분석에 사용합니다.

CORS나 사이트 정책 때문에 본문을 가져오지 못하면 URL과 링크 텍스트만으로 fallback 분석을 요청합니다. 같은 URL은 background service worker 메모리에 캐싱해 중복 API 호출을 줄입니다.

## 2. 설치 방법

1. 이 폴더의 파일을 모두 준비합니다.
2. `options.html`에서 사용할 Groq 또는 Cerebras API Key를 준비합니다.
3. Chrome 확장 프로그램 관리 페이지에서 압축 해제된 확장 프로그램으로 로드합니다.

## 3. API Key 설정 방법

1. 확장 프로그램 관리 페이지에서 이 확장 프로그램의 `세부정보`를 엽니다.
2. `확장 프로그램 옵션`을 클릭합니다.
3. Groq API Key 또는 Cerebras API Key를 입력하고 `저장`을 누릅니다. 여러 개를 쓰려면 한 줄에 하나씩 입력합니다.

API Key는 `chrome.storage.local`에 저장됩니다. `content.js`에는 API Key가 들어가지 않으며, AI API 호출은 `background.js`에서만 수행됩니다. 단, 이 저장 값은 별도 암호화 없이 브라우저의 확장 프로그램 로컬 저장소에 보관되므로 개인 기기에서만 사용하는 것을 권장합니다. 한도 초과나 인증 실패 등으로 현재 키를 사용할 수 없으면 다음 키로 자동 재시도합니다.

## 4. 크롬 확장 프로그램 로드 방법

1. Chrome 주소창에 `chrome://extensions`를 입력합니다.
2. 오른쪽 위의 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
4. 이 프로젝트 폴더를 선택합니다.
5. 웹페이지를 새로고침한 뒤 뉴스 링크 위에 마우스를 올려 테스트합니다.

## 사용 흐름

1. 웹페이지의 링크 위에 마우스를 올립니다.
2. 1초 동안 같은 링크 위에 머무르면 확장 프로그램이 링크를 인식합니다.
3. background service worker가 URL과 링크 텍스트를 받고, 가능한 경우 본문 일부를 가져옵니다.
4. Groq 또는 Cerebras API로 먼저 뉴스 기사인지 판별합니다.
5. 뉴스 기사로 확인된 경우에만 신뢰도와 어그로도를 분석합니다.
6. 결과는 마우스 옆 팝업에 표시됩니다.
7. 확장 프로그램 아이콘을 누르면 링크 인식, 본문 수집, 뉴스 판별, 분석 완료 같은 현재 진행 상태를 확인할 수 있습니다.

## 5. 점수 계산 기준 설명

### 신뢰도 `credibility_score` 0~100점

- 출처 명확성 20점: 기자명, 언론사, 작성일, 공식 출처 존재 여부
- 제목/본문 일치도 25점: 제목이 본문 내용을 정확히 반영하는 정도
- 근거 충실도 25점: 통계, 공식 발표, 전문가 인용, 자료 출처 존재 여부
- 표현 중립성 15점: 감정적·선동적 표현이 적은 정도
- 맥락 제공성 15점: 사건의 배경, 반대 의견, 한계, 추가 설명 제공 여부

### 어그로도 `clickbait_score` 0~100점

높을수록 나쁜 점수입니다.

- 과장 표현 20점: “충격”, “경악”, “역대급”, “난리”, “소름” 같은 표현
- 궁금증 유도 20점: “알고 보니”, “이유는?”, “결과는?”, “정체는?” 같은 표현
- 제목/본문 불일치 25점: 제목이 본문보다 과장되거나 다른 인상을 주는 정도
- 감정 자극 20점: 분노, 공포, 혐오, 불안, 논란을 과도하게 유도하는 정도
- 핵심 정보 은폐 15점: 제목에서 중요한 주어·대상·결과를 숨기는 정도

## 6. 수행평가 발표용 설명

이 프로젝트는 사용자가 뉴스를 클릭하기 전에 링크의 신뢰도를 빠르게 참고할 수 있게 돕는 AI 기반 브라우저 확장 프로그램입니다.

구조는 보안을 고려해 세 부분으로 나뉩니다. `content.js`는 웹페이지에서 링크 hover와 팝업 UI만 담당합니다. `background.js`는 API Key를 읽고 Groq 또는 Cerebras API를 호출합니다. `options.html`과 `options.js`는 사용자가 API Key를 저장하고 삭제하는 설정 화면입니다.

Manifest V3 규칙에 맞춰 background service worker를 사용하며, API Key를 웹페이지에 노출될 수 있는 content script에 넣지 않았습니다. 또한 같은 URL을 반복 분석하지 않도록 캐시를 적용해 API 사용량을 줄였습니다.

## 7. 한계점

AI 판단은 참고용이며 최종 팩트체크가 아닙니다. 본문 추출은 사이트의 CORS 정책, 로그인 요구, paywall, 동적 렌더링 구조에 따라 실패할 수 있습니다. 또한 AI는 제공된 일부 텍스트만 보고 판단하므로 실제 기사 전체의 정확성을 완전히 보장하지 않습니다.
